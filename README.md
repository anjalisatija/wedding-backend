# Wedding Booking Backend (FastAPI)

## Features
- Create wedding bookings
- Fetch all bookings
- MongoDB integration
- Swagger UI for API testing

## Tech Stack
- FastAPI
- MongoDB
- Python

## API Endpoints
- GET / → Health check
- POST /book → Create booking
- GET /bookings → Fetch all bookings

## Swagger UI
Available locally at:
http://127.0.0.1:8001/docs
